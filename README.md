Draft Helper API (Генератор черновиков)

Сервис для автоматической генерации текстов писем и постов с использованием FastAPI и LLM.

Как запустить проект

Вариант 1: Через Docker (рекомендуется)
1. Установите Docker и Docker Compose.
2. Выполните команду в терминале:
   ```bash
   docker-compose up --build
Откройте документацию: http://127.0.0.1:8000/docs

Вариант 2: Локальный запуск
1. Установите зависимости: pip install -r requirements.txt
2. Запустите сервер: uvicorn main:app --reload
Откройте документацию: http://127.0.0.1:8000/docs


***Для запуска необходим VPN(специфика взятого API)***
