Draft Helper API (Генератор черновиков)

Это интеллектуальный сервис для автоматической генерации текстов (email-писем и постов для соцсетей). В основе проекта лежит **FastAPI** и нейросеть (Groq API).

---

Как запустить проект

1. **Установите Docker Desktop** с официального сайта.
2. **Откройте терминал** (командную строку) и перейдите в папку с проектом:
   cd путь/к/папке/с/проектом
3. Запустите сборку и запуск:
   docker-compose up --build
Дождитесь сообщения: Когда в консоли появится надпись Application startup complete, сервис готов.

Откройте в браузере: http://127.0.0.1:8000/docs

---

Важное примечание
Для работы нейросети **необходим включенный VPN**, так как используемое API (Groq) может быть недоступно напрямую из РФ.
